# pixi-coloring-figma

Тестовая площадка на Next.js + Pixi для быстрой проверки комплектов раскраски из Figma.

Что умеет сейчас:
- загружает набор сегментов из `public/...`
- собирает картинку по координатам
- позволяет красить сегменты палитрой
- показывает результат на desktop и mobile UI
- умеет работать как с готовым `layout.json`, так и с JSON координат из Figma через внутренний адаптер

Важно:
- это именно sandbox для проверки комплектов из Figma
- финальная логика прод-приложения сюда пока не закладывалась
- отдельный `stroke.png` не обязателен, если сегменты уже экспортированы с собственным контуром

Текущие режимы:
- `figma-ready` — готовый эталонный комплект
- `geometry` — пример набора без отдельного `stroke`

Как запустить:

```bash
npm install
npm run dev
```

Открыть:
`http://localhost:3000/demo-coloring`

Как добавлять новый комплект:

1. создать папку в `public/<dataset-name>/`
2. положить туда `seg*.png`
3. положить либо `layout.json`, либо JSON координат из Figma
4. добавить новый набор в `src/data/coloring/demoSets.ts`

Внутренний поток данных:

* внешний JSON из Figma
* адаптер проекта
* внутренний `ColoringLayout`
* сборка Pixi-сцены

Назначение проекта:
быстро проверять, как набор из Figma будет выглядеть и работать в браузере, особенно на мобильных устройствах.
